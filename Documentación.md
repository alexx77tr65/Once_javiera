# Documentacion

## Decisiones semanticas

En la pagina se usaron etiquetas semanticas para organizar mejor el contenido.

Se uso <header> para el encabezado de la pagina donde aparece el titulo.
La etiqueta <nav> se uso para los enlaces de navegacion.
La etiqueta <main> contiene el contenido principal.
La etiqueta <article> se uso para el articulo principal de la pagina.
La etiqueta <aside> se uso para informacion adicional como las categorias.
La etiqueta <footer> se uso para el pie de pagina.
